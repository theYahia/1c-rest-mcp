# /balance — Остатки из 1С

## Алгоритм

1. Вызови `get_catalogs` или `get_documents` для получения данных
2. Если указан склад/касса — отфильтруй
3. Покажи остатки в табличном формате

## Формат ответа

```
## Остатки 1С

| Объект | Тип | Остаток |
|--------|-----|---------|
| Основной склад | Товары | 1 523 позиции |
| Касса | Денежные | 150 000.00 ₽ |

**Обновлено**: 31.03.2026
```

## Примеры

```
/balance
/balance Основной склад
```
